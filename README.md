## 使用指南

### 基础功能
- 回复 `0` 或 `菜单` - 显示功能菜单
- 回复 `1` 或 `我的植物` - 查看植物列表
- 回复 `2` 或 `养护知识` - 查看养护技巧
- 回复 `3` 或 `关于我们` - 查看项目信息

### 植物管理
- `添加 植物名称` - 添加新植物（例如：添加 绿萝）
- `删除 植物名称` - 删除植物（例如：删除 绿萝）
- 查看列表会显示所有植物和添加日期

### 示例对话
```
用户: 添加 绿萝
系统: ✅ 添加成功！植物名称：绿萝

用户: 添加 多肉植物
系统: ✅ 添加成功！植物名称：多肉植物

用户: 1
系统: 🌿 我的植物（共2株）
      1. 绿萝 📅 2026-02-09
      2. 多肉植物 📅 2026-02-09

用户: 删除 绿萝
系统: ✅ 已删除植物：绿萝
```
